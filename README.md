# CS 360: Mobile Architecture & Programming

Course workspace for CS 360 (SNHU). It mirrors the course's Learning Modules:
each module folder holds one subfolder per graded item, numbered the same way the
course numbers them (for example, `3-2-Assignment-Android-Studio-Introduction`).

## How this is organized

- Each **module** is a folder (`Module-1` … `Module-8`).
- Inside a module, each **course item** gets its own folder, named to match the
  course (`X-1` participation activities, `X-2` assignment, `X-3` project, plus a
  resources folder where the module has one).
- Written assignments are kept in two formats: the original **Word document
  (`.docx`)** for submission and a **Markdown (`.md`)** version so the work reads
  directly on GitHub.
- Figures used in a document live in a **`res/`** folder next to it — Android
  Studio's resource-folder convention, since this is an Android Studio course.
